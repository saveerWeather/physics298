  • In order to describe motion in more than one dimension it is convenient to introduce the concept of Vectors, which take into account both the magnitude and direction of certain quantities.

  • A vector quantity must be specified by both a magnitude and direction.
          Examples are, velocity, force, displacement and acceleration .

          Contrast with scalar quantities which require only a magnitude.
          Examples of scalars are, mass, speed, time and distance.

          N.B.    The magnitude of an object's instantaneous velocity is its instantaneous speed.
                      Displacement is measured directly from the starting point.

  • Addition and subtraction

    • Subtraction is the addition of a "negative" vector.  "Negative" vector is opposite in direction with same magnitude as the original vector.

    • Graphical

    • Resolution.  "Reverse" of addition.  Any vector can be "resolved" into two (or more) components along perpendicular axes (x,y,z).  The vector can then be written in terms of the unit vectors i, j, k.  For example A = 6i + 2j + 3k represents a vector with components (6,2,3) which can be represented by a line from the origin to the point (6,2,3).

    • Vector addition (subtraction) via components
      1. Resolve each vector into its x,y (z) components
      2. Add (subtract) x and y components separately
      3. Combine x and y components to give resultant vector
